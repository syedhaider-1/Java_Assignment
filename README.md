🎬 Movie Ticket Booking System (Java)
📌 Overview

The Movie Ticket Booking System is a console-based Java application that allows users to view available movies, select show timings, book tickets, and manage bookings. The project is designed using modular programming with multiple classes (5–7 files) to demonstrate object-oriented principles.

🚀 Features
🎥 View list of available movies
🕒 Select show timings
🎟️ Book tickets
💺 Seat selection (basic simulation)
❌ Cancel bookings
📋 Display booking details
🛠️ Technologies Used
Java (Core Java)
OOP Concepts (Classes, Objects, Encapsulation)
File Handling / In-memory data structures (depending on your implementation)
